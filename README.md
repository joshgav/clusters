# Clusters

Notes, scripts, and other helpers for deploying Kubernetes clusters.

## Install and configure and OpenShift cluster

1. Install openshift-install, aws, oc, kubectl, jq, etc.
1. Install OpenShift on AWS using IPI ([link](../openshift/aws/ipi/))
1. Install cert-manager and replace ingress certs ([link](../configuration/certificates/))
1. Enable user authentication via Google OIDC ([link](../configuration/iam/))
1. Enable machine autoscalers ([link](../configuration/autoscaler/))
1. Enable user workload monitoring ([link](../configuration/monitoring))
1. Install GitOps ([link](../configuration/gitops))
